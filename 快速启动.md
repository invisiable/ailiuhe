# 幸运数字预测系统 - 快速启动

## ✅ 项目已完成设置！

### 📊 训练数据特点

- **数据字段**:
  - `date`: 日期 (2024-01-01 至 2024-10-26)
  - `number`: 幸运数字 (1-49范围)
  - `animal`: 12生肖 (鼠、牛、虎、兔、龙、蛇、马、羊、猴、鸡、狗、猪)
  - `element`: 五行 (金、木、水、火、土)

- **数据量**: 300条历史记录
- **文件位置**: `data/lucky_numbers.csv`

### 🚀 三种使用方式

#### 方式1: 快速测试（命令行）
```powershell
python test_predictor.py
```
直接看到预测结果，包含未来5期的幸运数字、生肖、五行。

#### 方式2: 图形界面（推荐）
```powershell
python main.py
```
打开GUI界面，可视化操作，查看详细训练过程。

#### 方式3: 重新生成数据
```powershell
python generate_data.py
```
生成新的300条训练数据（如果想要不同的数据分布）。

### 🎯 模型性能指标

- **测试集MAE**: ~6.57 (平均误差约6-7个数字)
- **测试集RMSE**: ~7.85
- **测试集R²**: ~0.19

1. **首次使用**: 先运行 `test_predictor.py` 看看效果
