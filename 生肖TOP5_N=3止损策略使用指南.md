# 生肖TOP5止损策略使用指南（N=3优化版）

## 功能简介

生肖TOP5投注策略新增**策略选择功能**，用户可以根据风险偏好选择：
- 🛡️ **保守模式**：N=3止损策略（注重风险控制）
- 🐉 **激进模式**：纯倍投策略（追求最大收益）

## 使用方法

### 1. 启动分析
在GUI主界面点击 **"🐉 生肖TOP5投注"** 按钮

### 2. 选择策略
弹出对话框提供三个选项：

#### ✅ 是(Yes)：保守模式（推荐）
- **策略**：N=3止损策略
- **ROI**：22.92%
- **最大回撤**：185元
- **适合**：风险厌恶者、稳健投资者

#### ❌ 否(No)：激进模式
- **策略**：纯斐波那契倍投
- **ROI**：20.35%
- **总收益**：1510元（最高）
- **最大回撤**：1080元
- **适合**：风险承受力强、追求最大收益者

#### ⏸ 取消(Cancel)
返回主界面，不进行分析

## 策略详解

### 保守模式（N=3止损）

#### 工作原理
```
连续3期失败 → 触发止损 → 暂停投注3期 → 自动恢复 → 倍数重置为1倍
```

#### 核心优势
1. **回撤显著降低**：185元 vs 纯倍投1080元（降低83%）
2. **连败有效控制**：最多连败3期即止损
3. **心理压力小**：暂停期可以冷静观察
4. **风险可控**：最大投入40元（2倍）

#### 200期实测数据
```
总投入：3900元
总收益：+894元
ROI：22.92%
命中率：51.48%
暂停期数：31期（15.5%）
最大回撤：185元
最大连败：3期
```

### 激进模式（纯倍投）

#### 工作原理
```
斐波那契倍投（1,1,2,3,5,8,13...） → 持续投注 → 无止损限制
```

#### 核心优势
1. **收益最大化**：绝对收益1510元（最高）
2. **不错过机会**：每期都投注，不暂停

#### 潜在风险
1. **回撤较大**：1080元（保守模式的5.8倍）
2. **连败风险**：可能出现8期连败
3. **单期投入高**：最高可达680元

#### 200期实测数据
```
总投入：7420元
总收益：+1510元
ROI：20.35%
命中率：50.00%
暂停期数：0期
最大回撤：1080元
最大连败：8期
```

## 常见问题

### Q1: 为什么N=3止损ROI更高？
**A**: 因为止损避免了高倍投注的大额亏损，虽然绝对收益低，但投入成本也降低了47%，因此ROI更高。

### Q2: 保守模式会错过盈利机会吗？
**A**: 会。暂停期间如果命中会错过。但测试显示，避免的高倍亏损超过错过的盈利，整体ROI反而提升。

### Q3: 激进模式风险有多大？
**A**: 200期测试中最大回撤1080元，单期最高投入680元。需要有充足资金和心理准备。

### Q4: 哪个策略能达到50% ROI？
**A**: 两个策略都无法达到50% ROI目标。要达到50% ROI，需要将命中率从50%提升到64%（提高14个百分点），这需要优化预测模型，而非投注策略。

### Q5: 触发止损后多久恢复投注？
**A**: 暂停3期后自动恢复，倍数重置为1倍。例如：
- 第10期：连亏3期，触发止损
- 第11-13期：暂停投注（不管输赢）
- 第14期：自动恢复，1倍投注

## 技术细节

### N=3止损算法
```python
if consecutive_losses >= 3:
    # 触发止损
    pause_for_3_periods()
    
after_pause:
    # 恢复投注
    reset_multiplier_to_1()
```

### ROI计算公式
```
ROI = (总收益 / 总投入) × 100%
```
