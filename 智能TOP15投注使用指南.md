## 🎯 投注策略详解

### 核心策略：斐波那契倍投

**数列**: 1, 1, 2, 3, 5, 8, 13, 21, 34...

**规则**:
1. 初始倍数: 1倍（投注15元）
2. 命中后: 重置为1倍
3. 未中后: 按斐波那契数列递增
4. 达到限制: 根据档位限制最大倍数

### 示例

**情景1: 连续未中** (标准模式)
```
第1期: 1倍 (15元) → 未中 → 累计-15元
第2期: 1倍 (15元) → 未中 → 累计-30元
第3期: 2倍 (30元) → 未中 → 累计-60元
第4期: 3倍 (45元) → 未中 → 累计-105元
第5期: 5倍 (75元) → ✓命中 → 收益+160元 → 累计+55元
第6期: 1倍 (15元) → 重新开始
```

**情景2: 高胜率期** (标准模式)
```
第1期: 1倍 → ✓命中 → +32元
第2期: 1倍 → ✓命中 → +32元
第3期: 1倍 → ✓命中 → +32元
累计收益: +96元
```

## 💡 使用建议

### 推荐使用标准模式的理由

2. **风险可控**: 最大回撤810元，在可承受范围
3. **收益稳定**: 命中率37.5%，符合预期

- 初次使用该系统
- 资金有限（<2000元）
- 风险承受能力低
- 追求长期稳定收益

### 何时选择激进模式

- 对系统充分了解
- 资金充裕（>5000元）
- 风险承受能力高
- 追求最大化收益
- 能接受大额回撤

## ⚠️ 重要提示

### 风险警告

1. **历史表现不代表未来**
   - 42%ROI是基于历史数据
   - 未来表现可能有差异
   - 需持续监控和调整

2. **倍投风险**
   - 连续不中会导致投注额快速增长
   - 务必设置止损点
   - 不要超出自身承受能力

3. **资金管理**
   - 标准模式建议至少准备2000元
   - 激进模式建议至少准备5000元
   - 保守模式建议至少准备1000元

### 使用建议

1. **新手必读**
   - 首次使用建议选择标准模式
   - 先小额试验，积累经验
   - 理解倍投机制后再增加投入

2. **止损策略**
   - 设定每日/每周最大亏损额
   - 达到止损点立即停止
   - 避免情绪化追投

3. **盈利管理**
   - 设定盈利目标（如+500元）
   - 达到目标后适当降低风险
   - 定期提取部分盈利

## 📝 常见问题

### Q: 为什么标准模式最推荐？
**A**: 经过200期实战测试，ROI达到42.04%，同时风险可控，最大回撤810元，适合绝大多数用户。

### Q: 保守模式和激进模式的ROI差距不大，有必要冒险吗？
**A**: 激进模式ROI仅高2-3%，但回撤可能翻倍，性价比不高。除非你有充足资金和强大心理承受能力。

### Q: 能否达到50% ROI？
**A**: 根据测试，倍投策略优化空间有限，已接近理论上限。要达到50%需要提高预测准确率，而非调整倍投策略。

### Q: 如果连续不中怎么办？
**A**: 标准模式最多8期连续不中（200期测试中）。建议设置止损点，如连续10期或回撤超过预设金额时暂停。

### Q: 多久能回本？
**A**: 根据标准模式42%ROI和37.5%命中率，理论上平均3期左右可净盈利32元。但实际情况会有波动。
