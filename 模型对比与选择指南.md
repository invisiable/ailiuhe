## 🎯 如何选择模型

### 按使用目的选择

#### 1. 追求高命中率 🏆
**推荐组合**: 生肖TOP6 + TOP15
```python
from zodiac_top6_predictor import ZodiacTop6Predictor
from top15_predictor import Top15Predictor

zodiac = ZodiacTop6Predictor()
top15 = Top15Predictor()

# 获取预测
zodiac_nums = set(zodiac.predict()['top18_numbers'][:12])
top15_nums = set(top15.predict_top15(...)[:15])

# 取交集（最准确）
best_picks = list(zodiac_nums & top15_nums)
```

**优势**:
- 两个模型都命中的号码准确率最高
- 通常能得到5-8个高质量号码
- 命中率可能达到60%+

#### 2. 追求大覆盖面 📈
**推荐组合**: 生肖TOP6 + TOP20
```python
from zodiac_top6_predictor import ZodiacTop6Predictor
from top20_predictor import Top20Predictor

zodiac = ZodiacTop6Predictor()
top20 = Top20Predictor()

# 获取预测并合并
zodiac_nums = zodiac.predict()['top18_numbers'][:15]
top20_nums = top20.predict(...)[:20]

# 合并去重
all_picks = list(set(zodiac_nums + top20_nums))
```

**优势**:
- 最大覆盖面（可达25-30个号码）
- 提高命中机会
- 适合愿意投入更多选择的用户

#### 3. 快速预测 ⚡
**推荐模型**: 生肖TOP6（单独使用）
```python
from zodiac_top6_predictor import ZodiacTop6Predictor

predictor = ZodiacTop6Predictor()
result = predictor.predict()

# 直接使用TOP6生肖和TOP12号码
zodiacs = [z for z, s in result['top6_zodiacs'][:4]]
numbers = result['top18_numbers'][:12]
```

**优势**:
- 一个模型即可
- 预测速度快
- 号码命中率45%+（实测）

#### 4. 生肖角度分析 🐉
**推荐模型**: 生肖TOP6
```python
predictor = ZodiacTop6Predictor()
result = predictor.predict()

# 先看生肖，再选号码
for zodiac, score in result['top6_zodiacs'][:3]:
    numbers = predictor.zodiac_numbers[zodiac]
    print(f"{zodiac}: {numbers}")
```

**优势**:
- 生肖维度更直观
- 便于理解和记忆
- 可以根据生肖选号

## 💡 推荐策略

### 新手策略
1. 使用生肖TOP6单独预测
2. 选择TOP2-3生肖
3. 每个生肖选1-2个号码
4. 共6-9个号码

### 进阶策略
1. 生肖TOP6 + TOP15组合
2. 取两个模型的交集
3. 优先选择生肖TOP2的号码
4. 共8-12个号码

### 专家策略
1. 同时运行3个模型：
   - 生肖TOP6
   - TOP15
   - TOP20（可选）
2. 多层过滤：
   - 第一层：所有模型都推荐的号码（最高优先级）
   - 第二层：2个模型推荐的号码（高优先级）
   - 第三层：1个模型推荐但符合其他条件的号码
3. 按优先级选择10-15个号码
