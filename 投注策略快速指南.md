### 策略对比表格

```
策略对比:
├─ 马丁格尔(激进): 总收益+245元, ROI +12%, 最大回撤-68元
├─ 斐波那契(稳健): 总收益+186元, ROI +10%, 最大回撤-52元
```

**解读**:
- **总收益**: 100期累计盈亏（越高越好）
- **ROI**: 投资回报率（越高越好）
- **最大回撤**: 最大亏损金额（越小风险越低）

### 下期投注建议

```

【收益预期】

  风险等级: 低风险 ✓
```

**操作建议**:
1. 购买预测的TOP5数字
2. 每个数字买2元（共10元）
3. 如果命中，下期恢复1倍
4. 如果未中，下期按新建议调整

```
开始

  │

  │   │   ├─ 是 → 按建议投注
  │   │   └─ 否 → 降低倍数或暂停
  │   │
  │   └─ 检查止损条件
  │       ├─ 连续亏损>5期？ → 暂停
  │       ├─ 累计亏损>预算？ → 停止
  │       └─ 否 → 继续
  │
  ├─→ 记录结果
  │   ├─ 命中 → 重置计数，1倍投注
  │   └─ 未中 → 更新亏损，等待建议
  │
  └─→ 下一期
```

## 🔧 自定义参数

编辑配置（可选）:

```python
# 在betting_strategy.py中修改
betting = BettingStrategy(
    base_bet=5,        # 基础5元
    win_reward=45,     # 命中奖励45元
    loss_penalty=15    # 未中惩罚15元
)

# 调整最大倍数
betting.max_multiplier = 10  # 最大10倍，可改为5、15、20等
```

## ⚠️ 重要提醒

### 资金准备

不同最大倍数需要的资金储备：
- 5倍: 约75元
- 10倍: 约250元
- 20倍: 约1000元

### 风险控制

设置个人止损线：
- 单期亏损上限
- 累计亏损上限  
- 连续亏损次数上限

### 心态管理

- ✓ 严格按策略执行
- ✓ 不因连续亏损而恐慌
- ✓ 不因连续盈利而贪婪
- ✗ 避免情绪化决策
- ✗ 不超出承受能力
