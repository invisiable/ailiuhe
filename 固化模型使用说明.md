# 固化版混合组合策略预测器 v1.0

## 模型简介

本模型基于历史数据验证，采用混合组合策略，在最近10期验证中达到50%的TOP15预测成功率。

### 核心策略
- **TOP 1-5**：使用最近10期数据策略（精准预测）
- **TOP 6-15**：使用全部历史数据策略（稳定覆盖）

### 验证成绩
- TOP 15 成功率：50.0%
- TOP 10 成功率：50.0%
- TOP 5 成功率：20.0%

## 快速使用

### 方式1：直接运行预测
```bash
python final_hybrid_predictor.py
```

### 方式2：在代码中使用
```python
from final_hybrid_predictor import FinalHybridPredictor

# 创建预测器
predictor = FinalHybridPredictor()

# 生成预测
top15 = predictor.predict()
print(f"TOP15预测: {top15}")

# 获取详细信息
info = predictor.get_prediction_info()
print(f"最新一期: {info['latest_period']}")
print(f"最近10期: {info['recent_10_numbers']}")
```
