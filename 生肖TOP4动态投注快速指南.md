### 倍数计算逻辑
```python
def calculate_dynamic_multiplier(consecutive_wins, consecutive_losses):
    if consecutive_wins > 0:
        # 连胜：每胜1期减少0.5倍（最低0.5倍）
        multiplier = max(0.5, 1.0 - consecutive_wins * 0.5)
    elif consecutive_losses == 1:
        multiplier = 2.0  # 首败+1倍
    elif consecutive_losses == 2:
        multiplier = 4.0  # 连败2期+3倍
    else:
        multiplier = 4.0 + (consecutive_losses - 2) * 2  # 每期再+2倍
    
    return min(multiplier, 10.0)  # 最大10倍限制
```
